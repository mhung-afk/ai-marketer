## DynamoDB Schema

### Table: HealingBedroomContent (Phase 1, Extended)

**Primary Key**: `item_id` (Partition Key, type: String, GUID)

**Attributes**:

| Attribute | Type | Purpose | Notes |
|-----------|------|---------|-------|
| **item_id** (PK) | String | Unique content identifier | Generated on first ingest via `src/common/utils.generate_item_id()` |
| **source_platform** | String | Social media source | Enum: `tiktok`, `instagram`, `facebook` |
| **source_url** | String | Original post URL | Used for dedup verification; example: `https://www.tiktok.com/@user/video/123456` |
| **source_post_id** | String | Platform-specific post ID | For platform-native lookups; example: TikTok video ID |
| **original_caption** | String | Original post caption (English/platform language) | Max 2000 chars; raw text scraped from platform |
| **ai_caption_vi** | String | AI-generated Vietnamese caption | Generated by Claude; max 300 chars; includes hashtags |
| **ai_caption_tone** | String | Tone applied to caption | Enum: `calming`, `motivational`, `aesthetic`, `aspirational` |
| **s3_image_key** | String | S3 object key for stored image | Format: `raw/YYYY-MM-DD/HH-MM-SS-GUID.jpg` |
| **cloudfront_url** | String | Public CloudFront distribution URL | Format: `https://d123.cloudfront.net/raw/YYYY-MM-DD/...` |
| **content_hash** | String | MD5 hash of image file | 32-char hex string; used for dedup via GSI |
| **image_size_bytes** | Number | Downloaded image size | For tracking bandwidth |
| **image_width** | Number | Image dimensions (pixels) | For frontend rendering hints |
| **image_height** | Number | Image dimensions (pixels) | For frontend rendering hints |
| **status** | String | Processing status | Enum: `RAW` (initial), `APPROVED` (Phase 3), `PUBLISHED` (Phase 3), `REJECTED` (Phase 3), `ERROR` (failed processing) |
| **error_reason** | String (Optional) | If status=ERROR, reason for failure | Example: `"Claude timeout after 3 retries"` |
| **retry_count** | Number | Number of processing attempts | 0 on first attempt; incremented on retry |
| **created_at** | String | ISO8601 timestamp when item first ingested | Format: `2026-05-06T14:30:45Z` |
| **processed_at** | String | ISO8601 timestamp when caption generated | Format: `2026-05-06T14:31:02Z` |
| **updated_at** | String | Last update timestamp | Tracks when status/fields updated |
| **metadata** | Map (JSON) | Additional unstructured data | Keys: `source_engagement` (likes, comments), `source_author`, `source_location`, `campaign_tag` |
| **ttl** | Number (Optional) | DynamoDB TTL for auto-cleanup | Epoch seconds; set to now + 90 days for error items |

### Global Secondary Index: GSI_ContentHash

**Purpose**: Enable O(1) deduplication lookup by image content hash

| Attribute | Type | Role |
|-----------|------|------|
| **content_hash** | String | Partition Key (GSI) |
| **created_at** | String | Sort Key (GSI) |
| **source_url** | String | Projection: INCLUDE (needed for dedup verification) |
| **item_id** | String | Projection: INCLUDE (needed to link to original item) |

**Query Pattern**:
```python
# Check if image already processed
response = dynamodb.query(
    TableName='HealingBedroomContent',
    IndexName='GSI_ContentHash',
    KeyConditionExpression='content_hash = :hash',
    ExpressionAttributeValues={':hash': {'S': 'a1b2c3d4e5f6...'}}
)
if response['Items']: return "DUPLICATE_FOUND"
```

## Pydantic Models (Python Type Safety)

**File**: `src/common/schemas.py`

```python
from pydantic import BaseModel, Field, validator
from typing import Optional, Dict, Any
from enum import Enum
from datetime import datetime

class SourcePlatform(str, Enum):
    TIKTOK = "tiktok"
    INSTAGRAM = "instagram"
    FACEBOOK = "facebook"

class ContentStatus(Enum):
    RAW = "RAW"
    APPROVED = "APPROVED"
    PUBLISHED = "PUBLISHED"
    REJECTED = "REJECTED"
    ERROR = "ERROR"

class CaptureTone(str, Enum):
    CALMING = "calming"
    MOTIVATIONAL = "motivational"
    AESTHETIC = "aesthetic"
    ASPIRATIONAL = "aspirational"

class ContentItem(BaseModel):
    """DynamoDB ContentItem record"""
    item_id: str = Field(default_factory=lambda: str(uuid4()))
    source_platform: SourcePlatform
    source_url: str
    source_post_id: Optional[str] = None
    original_caption: str
    ai_caption_vi: Optional[str] = None
    ai_caption_tone: Optional[CaptureTone] = None
    s3_image_key: Optional[str] = None
    cloudfront_url: Optional[str] = None
    content_hash: Optional[str] = None
    image_size_bytes: Optional[int] = None
    image_width: Optional[int] = None
    image_height: Optional[int] = None
    status: ContentStatus = ContentStatus.RAW
    error_reason: Optional[str] = None
    retry_count: int = 0
    created_at: datetime = Field(default_factory=datetime.utcnow)
    processed_at: Optional[datetime] = None
    updated_at: datetime = Field(default_factory=datetime.utcnow)
    metadata: Dict[str, Any] = Field(default_factory=dict)
    ttl: Optional[int] = None
    
    @validator('original_caption')
    def caption_not_empty(cls, v):
        if not v or len(v.strip()) == 0:
            raise ValueError("original_caption cannot be empty")
        return v
    
    @validator('ai_caption_vi')
    def vietnamese_caption_length(cls, v):
        if v and len(v) > 300:
            raise ValueError("ai_caption_vi must be ≤ 300 characters")
        return v

class IngestionRun(BaseModel):
    """Represents a single ingestion execution"""
    run_id: str = Field(default_factory=lambda: str(uuid4()))
    start_time: datetime = Field(default_factory=datetime.utcnow)
    end_time: Optional[datetime] = None
    platform: Optional[SourcePlatform] = None
    items_scraped: int = 0
    items_deduplicated: int = 0
    items_processed: int = 0
    items_failed: int = 0
    errors: list = Field(default_factory=list)
    
    def duration_seconds(self) -> Optional[float]:
        if self.end_time:
            return (self.end_time - self.start_time).total_seconds()
        return None
```

## Deduplication Logic

### Algorithm: Content Hash + URL Comparison

```python
def check_duplicate(apify_item: dict, dynamodb_client) -> tuple[bool, Optional[str]]:
    """
    Check if image already processed.
    
    Returns:
        (is_duplicate: bool, existing_item_id: str)
    """
    # 1. Download image from source URL
    image_bytes = download_image(apify_item['image_url'])
    
    # 2. Compute MD5 hash
    content_hash = hashlib.md5(image_bytes).hexdigest()
    
    # 3. Query DynamoDB GSI for hash
    response = dynamodb_client.query(
        TableName='HealingBedroomContent',
        IndexName='GSI_ContentHash',
        KeyConditionExpression='content_hash = :hash',
        ExpressionAttributeValues={':hash': {'S': content_hash}},
        ProjectionExpression='item_id,source_url'
    )
    
    if response['Items']:
        # 4. If hash found, verify source_url didn't change (image reposted)
        existing_item = response['Items'][0]
        if existing_item['source_url'] == apify_item['source_url']:
            return True, existing_item['item_id']  # Same source, exact duplicate
        else:
            return True, existing_item['item_id']  # Content reposted, still duplicate
    
    return False, None  # No duplicate found
```

### Expected Accuracy
- **Hash collision risk**: MD5 collision probability negligible for images (<1 in 2^64)
- **Duplicate detection rate**: 99%+ (captures reposts with different URLs)
- **False negatives**: Only if image is heavily edited post-scrape (highly unlikely for this use case)

## Lifecycle & Retention

### Processing Lifecycle

```
INGESTION → PROCESSING → RAW → [PHASE 3] → APPROVED/REJECTED → PUBLISHED/ARCHIVED
   ↓           ↓           ↓
Initial       Add S3      Ready for
ingest        + caption    approval
              
↓ [ERROR]
MOVED TO SQS DLQ
(TTL: 4 days)
```

### Cleanup Policies

| Status | Retention | Cleanup | TTL |
|--------|-----------|---------|-----|
| RAW | Until Phase 3 approval | Manual or automatic after 30 days | 30 days (2592000 sec) |
| ERROR | 4 days (investigation window) | Auto-delete to SQS DLQ | 4 days (345600 sec) |
| PUBLISHED | Indefinite | Manual archive to S3 Glacier (Phase 3+) | N/A |
| REJECTED | 7 days | Auto-delete (TTL) | 7 days (604800 sec) |
